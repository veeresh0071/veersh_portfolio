# Veeresh Pujar - Portfolio Website

A professional, minimalist portfolio website showcasing my work as a Full-Stack Developer and AI Engineer, built with React.js and Framer Motion.

## Features

- Modern black & white design aesthetic
- Smooth Framer Motion animations throughout
- Fully responsive (mobile-first approach)
- Sticky navigation with active section highlighting
- Animated hero section with professional photo
- Project showcase with filtering and modal views
- Skills section with animated progress bars
- Contact form with validation
- Social media links in footer

## Tech Stack

- React.js (functional components with hooks)
- Framer Motion for animations
- CSS Modules for styling
- Modern ES6+ JavaScript

## Installation

```bash
npm install
```

## Development

```bash
npm start
```

Runs the app in development mode at [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
```

Builds the app for production to the `build` folder.
